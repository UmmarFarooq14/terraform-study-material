Terraform:
----------

	Terraform is a Infrastructure as code tool developed by Hashicorp using a declarative configuration language called Hashicorp Configuration Language.
------------------------------------------------------------------------------------------------------------------------------------------------------------

Comments in Terraform:
---------------------
--> Single line  -- #
--> Multi-line -- /* --- */


Common Use Cases:
-----------------
--> Cloud Infrastructure provisioning.
--> Multi-cloud Deployments.
--> Kubernetes cluster Management.
--> CI/CD Integration.
--> Diaster Recovery.	
--> Infrastructure Auditing & compliance.


What is Infrastructure as code:
-------------------------------
	Infrastructure as code is the Practice of managing and Provisioning the computing Infrastructure like Networks, Servers and Databases using machine-readable configuration files rather than the manual process.
------------------------------------------------------------------------------------------------------------------------------------------------------------

CLI Commands:
------------
--> Terraform init --- Initialize the terraform working directory.
--> Terraform plan --- generates & show the execution plan.
--> Terraform apply --- apply the changes required to reach the desired date.
--> Terraform destroy --- destroy the resources managed by terraform.
------------------------------------------------------------------------------------------------------------------------------------------------------------ 

Provider:
--------
	Provider are acts as plugins interface between terraform and system service (or) platforms like AWS, GCP and Azure.
------------------------------------------------------------------------------------------------------------------------------------------------------------
